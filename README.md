## 3. Installation Guide

### Step 1: Install the AI Brain
1.  Locate **`gold_vibe.onnx`** in your project folder.
2.  Open MT5 -> **File** -> **Open Data Folder**.
3.  Copy `gold_vibe.onnx` into **`MQL5\Files`**.

### Step 2: Install the Code
1.  Copy all **`.mqh`** files (Core, AI_Engine, Risk, etc.) into **`MQL5\Include`**.
2.  Copy **`XAU_Master_AI_Gatekeeper.mq5`** into **`MQL5\Experts`**.
3.  (Optional) Copy **`XAU_Master_Indicator.mq5`** into **`MQL5\Indicators`**.

### Step 3: Compile & Run
1.  Open **MetaEditor** (F4).
2.  Open `XAU_Master_AI_Gatekeeper.mq5` and press **F7** (Compile).
3.  Attach to **XAUUSD (Gold) M5 Chart**.
4.  **Important**: Ensure "Enable ONNX Vibe Check" is set to `true` in inputs.

## 4. Troubleshooting
- **"AI Error: Could not load model..."**: You forgot to put `gold_vibe.onnx` in `MQL5\Files`.
- **Strategy Tester freezes**: The DXY Monitor is disabled in tester mode to prevent this.
- **No Trades**: 
    - Check the Journal.
    - If "MacroBias: DXY pumping", the Dollar is too strong.
    - If "AI Vibe Check FAILED", the Neural Network hates the current volatility.
